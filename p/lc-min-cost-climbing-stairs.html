使用最小花费爬楼梯

题目：746. 使用最小花费爬楼梯。

为讨论方便，约定输入数组为 cost，数组长度为 n（开始那阶为第 0 阶，最后一阶为第 n-1 阶）。注意啊，cost[i] 表示一步爬上第 i 阶消耗的体力，这意味着，不管我们从第 i-1 阶跨上来，还是从第 i-2 阶跨上来，消耗的体力都是 cost[i]。

记爬上第 i 阶楼梯需要的最小体力为 dp[i]（提前暴露）这里举例解释一下。假如有一个这样的楼梯， cost = [10, 15, 20]，那么

                 楼顶
               ┏━━━━━━
               ┃ 20
          ┏━━━━┛
          ┃ 15
     ┏━━━━┛
  A  ┃ 10
━━━━━┛

开始时，我们站在 A 点，

  1. dp[0] 表示爬上第 0 阶消耗的最少体力。从地面爬上第 0 阶为 cost[0]，即 10，所以 dp[0] = 10；

  2. dp[1] 表示爬上第 1 阶消耗的最少体力。可以从地面直接跨上第 1 阶，消耗的体力为 cost[1]，即 15，所以 dp[1] = 15；

  3. dp[2] 表示爬上第 2 阶消耗的最少体力。爬上第 2 阶有两种方案：① 从第 0 阶跨一阶上去，② 从第 1 阶直接跨上去。我们关心最小的体力消耗，所以选择从第 0 阶往上跨（dp[0] < dp[1]），这时，爬上第 2 阶消耗的体力就是 dp[0] + cost[2] = 10 + 20，即 30，dp[2] = 30。

  4. 我们要爬上楼顶，要么从第 1 阶跨上去，要么从第 2 阶直接上去（这个楼梯有点奇怪啊，我站在最后一阶不算楼顶？？），所以爬到顶的消耗的最小体力为 dp[1] 和 dp[2] 的最小值，即 15。

注意，dp[1] 可不是爬完前两阶的最小花费，爬完前两阶的最小花费是 10，即从第 0 阶直接爬上去。

有了上面的分析，我们知道 dp 数组前两项就是 cost 数组的前两项，它满足递推关系 dp[i] = min(dp[i-1, dp[i-2]]) + cost[i]，最终的答案是 min(dp[n-1], dp[n-2])。这样，就可以写出以下代码：

/**
 * @param {number[]} cost
 * @return {number}
 */
var minCostClimbingStairs = function(cost) {
    const dp = [cost[0], cost[1]]
    for(let i = 2; i < cost.length; i++) {
        dp[i] = Math.min(dp[i-1], dp[i-2]) + cost[i]
    }
    return Math.min(dp[cost.length - 1], dp[cost.length - 2])
};

根据递推公式，我们只使用 dp 数组的最后两项，所以可以把 dp 数组前面多余的空间花费省掉。这里不优化了。
