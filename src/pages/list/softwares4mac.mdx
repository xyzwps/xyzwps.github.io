## 效率

- [Ice](https://github.com/jordanbaird/Ice): 顶部工具条管理工具。
   ```sh
   brew install --cask jordanbaird-ice
   ```
- [Maccy](https://github.com/p0deje/Maccy): 一个轻量级剪切板管理工具。
   ```sh
   brew install maccy
   ```
- [LinearMouse](https://linearmouse.app/zh-HK/): 修改鼠标指针速度的工具。
  ```sh
  brew install --cask linearmouse
  ```

## 开发

- [Homebrew](https://brew.sh/): 一个 Mac 上的包管理工具。
- [iTerm](https://iterm2.com/): 一个终端模拟器。
