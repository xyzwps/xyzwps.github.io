区域和检索 - 数组不可变

题目303. 区域和检索 - 数组不可变。

这一题非常简单，用前缀和就可以直接撸出来了。举个例子：

  • 输入数组为 nums = [-2, 0, 3, -5, 2, -1]

  • 前缀和为 prefixSum = [-2, -2, 1, -4, -2, -3]

  • 那么

    • sumRange(0, 2) = prefixSum[2] - prefixSum[0] + nums[0] = 1

    • sumRange(2, 5) = prefixSum[5] - prefixSum[2] + nums[2] = -1

代码也超级简单不写了。
