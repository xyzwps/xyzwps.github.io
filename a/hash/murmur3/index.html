Murmur3 Hash

介绍

Murmur 哈希是一种具有低碰撞率和高性能的现代非加密型哈希函数。 由 Austin Appleby 在 2008 年发明。 因此，它适合于一般的基于哈希的查找，而不适合于加密使用。 名称由乘法(MUltiply)和旋转(Rotate)组成，这两种操作在其内部循环中使用。 所有 MurmurHash 版本都是公共域软件。

Murmur3 只是第三个版本，也是当前最新版本。Murmur3 哈希函数是当今最好的哈希函数之一，具有优异的性能。 它有两种变体: 产生 32 位哈希值的 Murmur3A 和产生 128 位哈希值的 Murmur3F。

现成的实现

在 Java 界，Apache Commons Codec 中有现成的实现， Guava 中也有现成的。

用途

  • 生成大致唯一的值
  • 哈希表
  • TF（戳这里见对 TF 的解释）
  • HyperLogLog
  • 布隆过滤器
  • 一致性哈希
