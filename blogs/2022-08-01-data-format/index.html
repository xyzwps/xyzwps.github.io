一些数据格式

  • JSON5 is a superset of JSON that aims to alleviate some of the limitations of JSON by expanding its syntax to include some productions from ECMAScript 5.1.
  • Typesafe Config: configuration library for JVM languages using HOCON files.
  • RON is a simple readable data serialization format that looks similar to Rust syntax. It’s designed to support all of Serde’s data model, so structs, enums, tuples, arrays, generic maps, and primitive values.
  • YAML is a human-friendly data serialization language for all programming languages.
  • Avro is a data serialization system.
  • CBOR: RFC 8949 Concise Binary Object Representation
