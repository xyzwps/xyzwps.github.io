## 第一个程序

按照惯例，学习一门编程语言的第一个程序都是 Hello world。下面是 Java 版本：

```java
// HelloWorld.java
class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}
```

Java 语言是一种编译型语言，所以需要先对其进行变编译：

```sh
javac HelloWorld.java
```

变异后得到 `HelloWorld.class` 文件。之后就可以通过 Java 虚拟机（Java virtual machine, JVM）来执行这个 `.class` 文件了：

```sh
java HelloWorld # 执行 HelloWorld.class
```

## 启动单文件源代码程序

从 JDK 11 开始，`java` 命令支持直接运行单文件源代码程序。

```bash
java HelloWorld.java
```

见 [JEP 330](https://openjdk.org/jeps/330)。

## 启动多文件源代码程序

从 JDK 22 开始，`java` 命令支持直接运行多文件源代码程序。

```java
// HelloWorld2.java
class HelloWorld2 {
  public static void run() {
    System.out.println("Hello, World!");
  }
}

// HelloWorld.java
class HelloWorld {
  public static void main(String[] args) {
    HelloWorld2.run();
  }
}
```

然后只要执行以下命令就可以了：

```bash
java HelloWorld.java
```

对于预编译的类（如 class 文件、jar 包）等的支持，巷见 [JEP 458](https://openjdk.org/jeps/458)。
