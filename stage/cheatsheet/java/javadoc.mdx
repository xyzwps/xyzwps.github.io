## Markdown 文档注释

从 Java 23 开始，Java 支持使用 Markdown 文档注释。Markdown 文档注释用 `///` 标记。

下图中左边是传统文档注释，右边是 Markdown 文档注释。

![传统文档注释和 Markdown 文档注释的区别](/assets/img/cheatsheet/java/md-comments-diff.png)

Markdown 注释支持 CommonMark 标准，简单的 GFM 表格也是支持的。

见 [JEP 467](https://openjdk.org/jeps/467)。
